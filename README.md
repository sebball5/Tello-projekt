# Tello-projekt
Tello projekt, med Sander, Sebastian og Sylvester.

LINK til miro 

https://miro.com/app/board/uXjVHb_tvd0=/ 


Vi starter med at finde ud af hvad den kan, for så at kunne finde ud af hvad der kunne være sjovt

egenskaber: 
  Roll, Pitch og yaw
  Left, right, forward, back, up and down
  rotere med og mod uret
  kan flyve ud fra koordinater, i en bane eller kurve
  den kan flip 



  ### Problemformulering
  - Vi vil meget gerne lave en nemmere måde at styre dronen på, da den lige nu skal programeres, det vil vi gerne ændre til en mere brugervenlig overflade.
